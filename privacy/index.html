プライバシーポリシー

施行日：2024年1月1日
最終更新日：2024年7月21日

Paradise Lost Developer Team（以下「当チーム」といいます）は、Aivis-chan Bot（以下「本サービス」といいます）をご利用いただくユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本サービスにおける個人情報の取り扱いについて説明いたします。

1. 収集する情報

1.1 自動的に収集される情報

  • Discord ユーザーID
  • Discord サーバーID
  • コマンド使用ログ（日時、使用コマンド）
  • 音声合成リクエスト回数
  • エラーログ（トラブルシューティング目的）

1.2 ユーザーが提供する情報

  • 音声合成用テキスト
  • Bot設定情報（話者設定、音声パラメータなど）
  • サポート問い合わせ内容

2. 情報の利用目的

収集した情報は、以下の目的で利用いたします：

  • 本サービスの提供・運営
  • 音声合成機能の実行
  • サービスの改善・最適化
  • 技術的問題の診断・解決
  • 利用統計の作成（匿名化処理後）
  • サポートの提供
  • 利用規約違反の防止・対応

3. 情報の保存期間

3.1 音声合成テキスト

音声合成のために送信されたテキストは、音声合成処理完了後、直ちに削除されます。永続的な保存は行いません。

3.2 ログ情報

コマンド使用ログやエラーログは、サービス改善のため最大30日間保存されます。その後、自動的に削除されます。

3.3 設定情報

ユーザーのBot設定情報は、サービス利用期間中保存され、ユーザーがBotを削除した場合に削除されます。

4. 情報の共有・開示

当チームは、以下の場合を除き、収集した情報を第三者に提供することはありません：

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • 公的機関からの要請がある場合
  • 当チームの権利や財産を保護するために必要な場合
  • 緊急時において人の生命、身体または財産の保護のために必要な場合

5. セキュリティ

当チームは、収集した情報の安全性を確保するため、以下の対策を講じています：

  • データの暗号化
  • アクセス制限の実施
  • 定期的なセキュリティ監査
  • 従業員への教育・研修

6. Cookie等の技術

本サービスのWebサイトでは、サービス向上のためにCookieやGoogle Analyticsを使用する場合があります。これらの技術により収集される情報は、統計的な分析にのみ使用され、個人を特定するものではありません。

7. 第三者サービス

本サービスでは、以下の第三者サービスを利用しています：

  • Discord API
  • Google Analytics（Webサイト）
  • Google AdSense（Webサイト）

これらのサービスには、それぞれ独自のプライバシーポリシーが適用されます。

8. ユーザーの権利

ユーザーには、以下の権利があります：

  • 個人情報の開示請求
  • 個人情報の訂正・削除請求
  • サービス利用の停止

これらの権利を行使される場合は、下記連絡先までお問い合わせください。

9. 未成年者の個人情報

当チームは、13歳未満の未成年者から意図的に個人情報を収集することはありません。13歳未満の方が本サービスを利用される場合は、保護者の同意が必要です。

10. プライバシーポリシーの変更

当チームは、本プライバシーポリシーを必要に応じて変更することがあります。重要な変更がある場合は、Webサイトやサポートサーバーにて事前に通知いたします。
