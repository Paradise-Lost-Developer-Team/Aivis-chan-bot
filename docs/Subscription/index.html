機能の詳細比較

🔊 読み上げ文字数の拡張

長いメッセージや詳細な説明も途切れることなく読み上げ可能です。無料版の200文字から、Pro版は400文字、Premium版は800文字まで拡張されます。

無料版: 200文字
Pro版: 400文字
Premium版: 800文字

📚 カスタム辞書の拡張

サーバー独自の言葉や固有名詞を登録できるカスタム辞書の上限を拡張。無料版30件から、Pro版100件、Premium版は無制限となります。

無料版: 30件
Pro版: 100件
Premium版: 無制限

🎛️ カスタム読み上げスタイル

音量、ピッチ、速度などを組み合わせた独自の読み上げスタイルを作成・保存できます。Pro版では3つ、Premium版では10個まで保存可能です。

無料版

デフォルトスタイルのみ

Pro版

3つのカスタムスタイルを保存可能

基本パラメータの調整

Premium版

10個のカスタムスタイルを保存可能

高度なパラメータも調整可能

スタイル間の切り替えが簡単

📝 音声履歴機能

読み上げた内容の履歴を保存し、後から検索・確認できる機能です。Pro版では直近100件、Premium版では500件まで保存されます。

⚙️ 辞書のインポート・エクスポート

Premium版限定機能として、カスタム辞書をCSVファイルでインポート・エクスポートできます。大量の辞書登録に便利です。

🎤 専用音声モデル

Pro版とPremium版では、無料版の6種類に加えて、それぞれ専用の音声モデルが利用できます。より多彩な表現が可能です。

よくある質問

既存のボットのプランをアップグレードできますか？

はい、既存のボットをそのままPro版やPremium版にアップグレードできます。購入後、自動的に機能が有効化されます。

サブスクリプションはいつでもキャンセルできますか？

はい、いつでもキャンセル可能です。キャンセル後は期間満了まで機能を利用できます。

支払い方法は何がありますか？

クレジットカード、PayPal、銀行振込に対応しています。

サーバーごとに支払いが必要ですか？

いいえ、一度購入すると最大5つのサーバーでPro版/Premium版機能を利用できます。

無料プランに戻ることはできますか？

もちろん可能です。サブスクリプション期間が終了すると自動的に無料プランに戻ります。

返金ポリシーはありますか？

ご購入後7日以内で、かつ機能をほとんど使用していない場合は返金対応可能です。詳細はサポートまでお問い合わせください。

Pro版とPremium版の機能は追加されますか？

はい、定期的に新機能を追加しており、有料プラン向けの機能も随時拡充しています。
